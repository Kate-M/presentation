Scalable Vector Graphics

What is SVG?

  • SVG stands for Scalable Vector Graphics
  • SVG is used to define vector-based graphics for the Web
  • SVG defines the graphics in XML format
  • SVG is a W3C recommendation
  • SVG integrates with other W3C standards such as the DOM and XSL

SVG Advantages

  • SVG images can be created and edited with any text editor
  • SVG images are scalable
  • Every element and every attribute in SVG files can be animated
  • SVG images can be searched, indexed, scripted, and compressed
  • SVG is an open standard

JS-manipulation

  				
//svg.js							 
window.onload=function() {
  var object = document.getElementById("pacmanObject");
  var svgDocument = object.contentDocument;
  var pac = svgDocument.getElementsByClassName("pacman");
  var pacText = svgDocument.getElementsByClassName("pacman-text");
  pac[0].setAttribute("fill", "#EBEC8E");
  pacText[0].setAttribute("stroke", "#EBEC8E");
  pacText[0].setAttribute("stroke-width", "1px");
  pacText[0].setAttribute("fill", "transparent");
};

  stroke-dasharray:1000;/*controls the pattern of dashes and gaps used to stroke paths*/
